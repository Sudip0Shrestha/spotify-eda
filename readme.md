## BRIEF ONE LINE SUMMARY
Exploratory data analysis of Spotify tracks to uncover trends in popularity, genres, artists, and release patterns.

---

## OVERVIEW
This project analyzes Spotify music data to understand patterns in track popularity, artist influence, genres, and song releases over time. The project includes data cleaning, transformation, and visualization to extract meaningful insights.

---

## PROBLEM STATEMENT
With thousands of tracks released each year, it becomes difficult to understand:
- Which artists dominate popularity?
- Which genres perform best?
- How explicit content affects popularity?
- How music trends evolve over time?

This project aims to answer these questions using data analysis.

---

## DATASET
The dataset contains Spotify track and artist information including:
- Track details
- Artist popularity & followers
- Genres
- Album information
- Release year
- Track popularity
- Duration and explicit flag

Total records: ~8,500 tracks.

---

## METHODS
- Data cleaning and preprocessing
- Handling missing values
- Feature engineering (year extraction)
- Grouping & aggregation analysis
- Genre expansion and analysis
- Trend analysis by year
- Data visualization

---

## KEY INSIGHTS
- Taylor Swift leads in total track popularity and followers.
- Non-explicit tracks contribute more to total popularity.
- Pop and Hip-Hop genres dominate popularity metrics.
- Song releases increased significantly after 2010.
- Artist popularity strongly correlates with track popularity.
